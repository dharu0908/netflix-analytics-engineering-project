# Netflix Analytics Engineering Project

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![dbt](https://img.shields.io/badge/dbt-1.8+-orange.svg)](https://getdbt.com)
[![Snowflake](https://img.shields.io/badge/Snowflake-Ready-blue)](https://snowflake.com)

A modern **Analytics Engineering** project using **dbt (data build tool)** to transform raw Netflix movie and TV show data into clean, analytics-ready models on **Snowflake**.

## 🎯 Project Overview

This project demonstrates end-to-end analytics engineering practices:
- Loading raw data into Snowflake
- Building staging models (cleaning & standardization)
- Creating marts with business logic
- Answering key business questions about Netflix content

**Goal**: Provide actionable insights on Netflix's content library, trends, ratings, and performance metrics.

## ✨ Features

- **Modern Data Stack**: dbt + Snowflake
- **Layered Architecture**: Staging → Bridge → Marts
- **Data Quality Tests** included
- **Business Questions** documented
- **Reusable Macros** and best practices

## 🚀 Quick Start

### Prerequisites

- Snowflake account
- dbt Core or dbt Cloud
- Git

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/dharu0908/netflix-analytics-engineering-project.git
   cd netflix-analytics-engineering-project
   ```

2. **Configure dbt**
   ```bash
   cd dbt_project
   cp profiles.yml.example ~/.dbt/profiles.yml   # Configure your credentials
   ```

3. **Install dependencies**
   ```bash
   dbt deps
   ```

4. **Run the pipeline**
   ```bash
   dbt run
   dbt test
   ```
